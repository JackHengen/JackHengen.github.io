Margin is like shrinking inwards -adds space around element but shrinks element
Padding is like expanding outwards -takes space around element and enlarges element
width and height are calculated in two different ways, either box-sizing:content-box or box-sizing:border-box
content box adds padding and border to width and height to get total width and height
border box accounts for padding and border and sets total height and width to the height and width you set
